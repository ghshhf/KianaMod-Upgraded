## 🎮 全新机制：HonkaiEnergy (崩坏能量)

- 战斗开始时拥有 0 层HonkaiEnergy
- 通过卡牌效果积累崩坏能量
- **三档阈值**：3/7/11 层 — 达到阈值时触发额外效果
- 部分终焉卡牌需要消耗崩坏能量释放
- **遗物血蔷薇**提供开局崩坏能量积累

### 三形态架势系统

| 形态 | 效果 |
|------|------|
| 🔥 **炎律** (FlameStance) | 提升火焰伤害 50%，灼烧效果增强 |
| 🌌 **空律** (VoidStance) | 虚数卡牌费用 -1，控场能力增强 |
| ✨ **终焉** (FinalityStance) | 首张牌费用为 0，终焉卡牌增伤 |

## 📥 安装使用

### 前置依赖

- Slay the Spire（游戏本体）
- [ModTheSpire](https://github.com/kiooeht/ModTheSpire/releases) 3.23.2+
- [BaseMod](https://github.com/daviscook477/BaseMod/releases)

### 安装步骤

1. 从 [Releases](https://github.com/ghshhf/KianaMod-Upgraded/releases) 下载最新 `kiana-mod-upgraded-v1.1.0.jar`
2. 复制到 `Steam/steamapps/common/SlayTheSpire/mods/` 目录
3. **（重要）** 如果已有 `kiana-mod.jar`，请覆盖之，或将原文件备份后替换
4. 启动 Slay the Spire，在 Mod 列表勾选 **KianaMod** 和 **BaseMod**
5. 点击 Play 开始游戏

### 切换脚本（推荐）

项目提供了 `build.bat` 一键构建脚本，用于从源码构建打包和本地部署：

```bash
# 构建（需要 Java 8 JDK）
build.bat
```

构建后会自动输出到 `dist/` 目录。

## 🔧 从源码构建

### 环境要求

- **JDK 21+**（使用 `--release 8` 交叉编译，兼容 StS 的 Java 8 运行时）
- **Python 3.6+**（用于打包脚本 `build_pack.py`）
- Slay the Spire 游戏文件（用于获取依赖 jar）

### 构建步骤

```bash
# 1. 克隆仓库
git clone https://github.com/ghshhf/KianaMod-Upgraded.git
cd KianaMod-Upgraded

# 2. 一键构建（编译 + 打包）
build.bat

# 3. 输出在 dist/ 目录
ls dist/
```

### GitHub Actions CI

每次推送到 `master` 分支时，CI 会自动验证构建是否通过：
- ✅ 编译所有 51 个源文件
- ✅ 执行 Java 8 兼容性验证
- ✅ 上传构建产物 (jar) 为 Artifact
